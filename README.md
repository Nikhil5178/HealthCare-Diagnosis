
# Healthcare Diagnosis with Multimodal Fusion:

## 📌 Project Description
A healthcare diagnosis system that accurately predicts diseases using patient data, improving diagnostic accuracy and supporting early detection.

## 🚀 Features
- Data preprocessing and analysis (EDA)
- Machine learning model for disease prediction
- Performance evaluation (accuracy, confusion matrix)
- Visualization of results

## 📊 Dataset
- Heart Disease Dataset (small and efficient)

## 🧠 Model
- Neural Network (TensorFlow/Keras)

## 📈 Results
- Achieved ~85-90% accuracy
- Improved prediction using multiple features
